Sobre o projeto

Gênesis RPG é uma plataforma criada para facilitar a vida de mestres e jogadores, reunindo em um só lugar fichas de personagem, campanhas, sistema de regras e controle de turnos. A proposta é tornar a organização das mesas mais simples, visual e integrada, permitindo que cada grupo foque no que realmente importa: contar boas histórias.

Este projeto foi pensado para ser flexível e expansível, servindo tanto para grupos que estão começando no RPG quanto para mesas veteranas que buscam uma ferramenta moderna para acompanhar suas aventuras.
